# Placement Predictor (ML Project)
This is a Machine Learning based web app that predicts whether a student will be placed or not based on CGPA and IQ.
## Tech Stack
- Python
- Scikit-learn
- Streamlit
## 📊 Features
- Takes CGPA and IQ as input
- Predicts placement using Logistic Regression
- Simple and interactive UI
#Dataset : placement.csv 
## How to Run
```bash
streamlit run app.py
